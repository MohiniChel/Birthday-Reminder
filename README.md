# Birthday-Reminder

Functionalities:-

1. Can record all the birthday's with the following details
   * Name
   * Age
   * Photo
  
2. Can show how many birthday's are there today.

3. Can clear all the completed birthdays.

#### Used Technologies : Reactj.s, React Hook.
